================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了时钟输出功能，可输出HSE波形。

================================================================================
测试环境：
测试用板：PY32M010_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. PA06引脚接入外部24MHz时钟；
3. 按下用户按键；
4. 可由示波器观察到PA07输出HSE波形（24MHz）。
