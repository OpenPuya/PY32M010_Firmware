================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了TIM1的输入捕获功能，配置PA1作为输入捕获引脚，PA1每检测到一个下降沿触
发捕获中断在捕获中断回调函数中翻转LED灯。

================================================================================
测试环境：
测试用板：PY32M010_STK
MDK版本： 5.28
IAR版本： 9.20

================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. 每次按下按键,LED灯翻转一次。
