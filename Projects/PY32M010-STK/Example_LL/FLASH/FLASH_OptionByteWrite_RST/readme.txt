================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了通过软件方式将RESET引脚改为普通GPIO。

================================================================================
测试环境：
测试用板：PY32M010_STK
MDK版本： 5.28
IAR版本： 9.20

================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 请将MCU断电并重新上电；
3. RST键变为GPIO功能，LED亮；
4. 注释掉"MODE_PC0 OB_SWD_PB6_GPIO_PC0",打开"MODE_PC0 OB_SWD_PB6_NRST_PC0",
编译重新下载到MCU;
5. 将MCU断电并重新上电；
6. RST键变为RESET功能，LED亮；
